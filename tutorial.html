MPI : Getting Started

Installing MPI on your laptop/workstation:
On ubuntu :
sudo apt-get install mpich2

To compile your code using MPICH you need to use this command:
mpicc filename.c

To run your code using MPI you need to use this command:
mpirun -np 6 ./a.out
